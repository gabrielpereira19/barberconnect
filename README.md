# BarberConnect 💈

Aplicativo mobile desenvolvido para facilitar o agendamento em barbearias, evitando filas e melhorando a experiência dos clientes.

## 🚀 Funcionalidades
- Agendamento de horários
- Lista de barbearias
- Tela de login e cadastro
- Favoritos
- Perfil do usuário

## 🛠 Tecnologias
- React Native
- Expo
- TypeScript
